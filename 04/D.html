Estas son las siguientes etiquetas de agrupacion

  1. div : etiqueta de agrupacion contenedora

  2. p : etiqueta de agrupacion de texto (parrafo)
  3. main : etiqueta de agrupacion de contenido principal de la pagina
  4. hr : etiqueta separadora de contenido agrupado y separa tematica del texto
  5. ol : etiqueta crea una lista numerada en orden
  6. ul : etiqueta crea una lista donde el orden no importa y no numerada
  7. li : Contiene uno de los items de la lista numerada o no numerada
  8. pre : Establece un texto preformateado
  9. blockquote : Agrupa informacion y caracteristicas de una cita
  10. dl : crea una lista de definiciones
  11. dt : Establece el termino de una definicion
  12. dd : Establece la desvripcion de un termino de una definicion
  13. figure : Agrupa un elemento visual en una figura o ilustracion
  14. figcaption : crea una leyenda a una figura o ilustracion para aclaracion de la figure
